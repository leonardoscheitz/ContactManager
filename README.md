# ContactManager

ContactManager é um projeto simples de gerenciamento de contatos feito em Java, usando Maven para dependências.

## Funcionalidades

- Adicionar contatos
- Listar contatos
- Buscar contatos
- Remover contatos

## Requisitos

- Java 8
- Maven

   cd ContactManage
